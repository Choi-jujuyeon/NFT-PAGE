# 만들고자 하는 것 

-> 일회용품을 줄이기 프로젝트 시작 ~~!~ 사람들은 희소성에 열광한다


-> 일회용품 컵 보증금제 시작
    보증금제도 : 일회용 컵에 담아 가면 보증금 300원을 더 내고
        컵을 반납하면 다시 돌려받는 일회용 컵

    ==> 플라스틱 컵 + 뜨거운 음료 담는 종이컵 해당됌
    ==> 길에 버려진 일회용 컵을 주워서 반환하는 경우에도
    보증금을 받을 수 있고 == 컵 표현에 한국 조폐공사 위변조 스티커 부착예정

    ==> 뚜레쥬르 ?ㅋㅋㅋㅋㅋ
    =>각 컵마다 바코드 인식이 가능한 POS기기에 컵 바코드 확인
    => 보증금 반환 방법 == 현금 or 계좌이체 가능함 
    
    반환되지 않는 컵이 더 많다
    == 그냥 버려지는 컵이 대다수 == 돈내고 컵을 사는 것밖에 안됌.
    == 현재 세종과 제주만 진행되고 있음
    == 회수율 끌어올리기 어려움 

    컵을 반환하려면 같은 프랜차이즈 매장으로 찾아가야함

바다로 흘러간 일회용 컵은 미세플라스틱으로 깨지거나 부서져
해양생태계를 위협하고 결국 우리에게 되돌아온다.
세계자연기금(WWF) 보고에 따르면 우리는 매주
신용카드 1장 무게의 플라스틱을 섭취한다고 한다.
요즘은 자원순환 기술이 발달해 다른 물질과 섞이지 않은 일회용 컵을
모으면 중요한 자원이 된다.
